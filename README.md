# CarRent

## Описание

**Почему был создан продукт?**
Я создал CarRent, чтобы уйти от бумажного учета и громоздких таблиц Excel в сфере проката автомобилей. Целью было сделать простой, надежный и понятный инструмент, который поможет сосредоточиться на работе с клиентами, а не на рутинных расчетах.

**Для чего был создан продукт?**
Приложение решает ключевые проблемы управления автопарком:
* Ведение единой базы данных автомобилей и клиентов.
* Автоматический расчет стоимости аренды в зависимости от количества дней.
* Контроль занятости автомобилей (исключение ситуаций, когда одну машину сдают дважды).
* Отслеживание технического состояния автопарка (статусы «В ремонте», «Свободен», «Аренда»).
* Проверка корректности данных при вводе (формат госномера, VIN, паспортные данные).

**Чем продукт лучше аналогов?**
В отличие от сложных автоматизированных рабочих мест (АРМ) и облачных систем, CarRent — это легковесное десктопное приложение. Оно работает локально с базой данных SQLite, не требует постоянного подключения к интернету и сложной настройки сервера. Интерфейс интуитивно понятен и не перегружен лишними функциями.

## Установка

Для установки и запуска приложения выполните следующие шаги:

1.  Убедитесь, что на вашем компьютере установлена Java (JDK) версии 17 или выше.
2.  Склонируйте репозиторий на локальную машину.
3.  Откройте проект в среде разработки (рекомендуется IntelliJ IDEA).
4.  Проект использует JavaFX. Убедитесь, что библиотеки JavaFX подключены корректно в настройках проекта или через систему сборки.
5.  База данных `carrent.db` будет создана автоматически при первом запуске приложения в корневой директории проекта.

## Использование

Инструкция для быстрого старта:

1.  **Автопарк**: Перейдите в раздел «Автопарк» и добавьте свои автомобили, нажав кнопку «+ Автомобиль». Заполните данные (Марка, Модель, Госномер, Цена).
2.  **Клиенты**: В разделе «Клиенты» внесите информацию о заказчиках. Система проверит правильность ввода телефона и паспортных данных.
3.  **Создание аренды**: Перейдите в «Журнал операций» и нажмите «+ Новая запись». Выберите клиента, автомобиль и даты аренды. Стоимость рассчитается автоматически.
4.  **Выдача и возврат**:
    * В журнале операций активные аренды можно завершить кнопкой «Забрать».
    * При возврате укажите состояние авто: «Все отлично» (машина вернется в парк) или «Есть повреждения» (машина отправится в статус «Ремонт»).
5.  **История**: В разделе «Завершенные заказы» или в карточке конкретного автомобиля можно посмотреть историю всех поездок.
